Security and safety are a top priority for Boppo.

The Boppo tablet does not have a microphone or camera.

Boppo firmware is signed and verified by the ESP32-S3 bootloader. The device's [HTTP server](/api) is protected by a bearer token which can be reset by performing a factory reset.

## Reporting

If you discover a security vulnerability, please report it to [security@boppo.com](mailto:security@boppo.com). Thank you for helping us keep Boppo secure.

If you would prefer to encrypt your report, you can encrypt using our [public SSH key](https://github.com/benhansen-io.keys) and [age](https://github.com/filosottile/age#encrypting-to-a-github-user).
