title: Hardware Specs
---

The Boppo tablet has:

* 10 custom mechanical keyboard switches with custom injection molded keycaps
* 4 individually addressable RGB LEDs per button (on the top, left, right, and bottom)
* a 3.5 Watt (max) speaker with an amplifier with plenty of headroom and advanced EQ
* an RFID antenna and reader that can read some IEC14443A/B tags (e.g. NFC 213)
* a 2,000 mAh rechargable LiPo battery
* 32 GB SD card
* a USB-C port (for charging, data commands and MIDI)
* a 3.5mm headphone jack
* ESP32-S3-N16R2 which is a microcontroller that has:
    * a dual-core CPU running at 240 Mhz
    * 512 kB SRAM
    * 2 MB PSRAM
    * 16 MB Flash
    * Wifi and Bluetooth communication
